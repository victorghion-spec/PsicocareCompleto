================================================================================
COMO USAR
================================================================================

1. Todos os locators estão definidos em: locators.robot
2. Keywords reutilizáveis estão em: resources/psicocare_resources.robot
3. Testes estão em: test_*.robot
4. Para melhor identificação, adicione testID aos componentes React Native
5. Veja GUIA_ADICIONAR_TESTID.md para instruções detalhadas

================================================================================
PRÓXIMOS PASSOS
================================================================================

1. Adicionar testID aos componentes React Native (ver GUIA_ADICIONAR_TESTID.md)
2. Atualizar locators.robot para usar testID ao invés de XPath
3. Executar testes: robot tests/robotframework/
4. Ajustar timeouts e locators conforme necessário
5. Configurar CI/CD para execução automática
